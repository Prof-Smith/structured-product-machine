# Structured Product Machine V5 Stable Teaching Flow

This version replaces the confusing 3D-style center component with a clearer teaching flow:

Market Return -> Structured Note Layers -> Investor Payoff

It keeps the working static GitHub Pages architecture and restores reliable button behavior.

## Upload
Upload these files directly to the repository root:

```text
index.html
README.md
css/styles.css
js/payoff.js
js/chart.js
js/app.js
```

GitHub Pages setting:

```text
Settings > Pages > Deploy from branch > main > /root
```
